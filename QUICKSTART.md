# iClippy Quick Start Guide

## Installation (3 Steps)

### Step 1: Build the App

Open Terminal and run:

```bash
cd /Users/bridan/Documents/iClippy
./build.sh
```

When prompted, press `y` to install to Applications folder.

### Step 2: Grant Permissions

1. Launch iClippy from Applications folder
2. macOS will ask for Accessibility permissions
3. Click **Open System Settings**
4. Toggle **iClippy** ON in Privacy & Security > Accessibility
5. Close System Settings

### Step 3: Start Using!

Press `⌘⇧V` (Command + Shift + V) to open iClippy.

## Quick Reference

### Open iClippy
- Press `⌘⇧V` anywhere

### Navigate
- `↑` `↓` arrow keys to select items
- `⏎` Enter to copy and auto-paste
- `⌫` Delete to remove item

### Features
- **Search**: Type to filter items
- **Pin**: Click pin icon to keep items forever
- **Clear**: Remove all unpinned items

## That's It!

iClippy now monitors your clipboard automatically. Every time you copy something, it's saved to your history.
